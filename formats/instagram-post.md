# Format: Instagram Post (1:1)

## Technische Daten

- **Seitenverhältnis:** 1:1 (quadratisch)
- **Empfohlene Auflösung:** 1080 x 1080 px
- **Modell-Parameter:** `aspectRatio: "1:1"`

## Kompositions-Regeln

- LUMI zentral oder leicht links-zentriert positioniert
- Hauptelement nimmt 50–60% der Fläche ein
- Annotation oben rechts oder unten links
- Kein Text am Rand (mind. 100px Abstand zum Bildrand)

## Prompt-Zusatz

```
square format, centered composition, Instagram feed optimized
```

## Einsatz

- Wissensposts
- Erklärungen
- Tipps & Tricks
- Zitate mit LUMI-Illustration
