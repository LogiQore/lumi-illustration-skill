# Format: YouTube Thumbnail (16:9)

## Technische Daten

- **Seitenverhältnis:** 16:9 (quer)
- **Empfohlene Auflösung:** 1280 x 720 px
- **Modell-Parameter:** `aspectRatio: "16:9"`

## Kompositions-Regeln

- LUMI links positioniert (aktive Seite)
- Rechte Seite: Platz für Titel-Text (wird außerhalb ergänzt)
- Starker visueller Kontrast – muss klein (Vorschau) erkennbar sein
- LUMI-Aktion muss auch bei 120px Breite noch lesbar sein

## Prompt-Zusatz

```
wide format, 16:9 ratio, YouTube thumbnail, subject on left side, right side space for text
```

## Einsatz

- YouTube Video-Thumbnails
- Video-Intros
- Kurs-Cover
- Webinar-Ankündigungen
