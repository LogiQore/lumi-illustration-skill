# Format: Pinterest Pin (2:3)

## Technische Daten

- **Seitenverhältnis:** 2:3 (vertikal)
- **Empfohlene Auflösung:** 1000 x 1500 px
- **Modell-Parameter:** `aspectRatio: "2:3"`

## Kompositions-Regeln

- LUMI im oberen Bereich (erste 60% der Höhe)
- Unterer Bereich: Platz für Titel-Text (wird außerhalb des Bildes ergänzt)
- Starke visuelle Aussage – Pinterest ist eine Suchmaschine!
- Annotation klar lesbar und themenbezogen

## Prompt-Zusatz

```
portrait format, 2:3 ratio, Pinterest optimized, strong visual hook in upper half
```

## Einsatz

- Tutorials & Anleitungen
- "So geht KI-Content"-Pins
- Listicle-Illustrationen
- Evergreen-Content für ki-content-creator.de
