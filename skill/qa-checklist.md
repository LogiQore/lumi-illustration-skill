# QA-Checkliste

Vor dem Speichern jedes generierten Bildes prüfen:

## Must-Have

- [ ] LUMI ist sichtbar und aktiv (nicht nur Dekoration)
- [ ] Hintergrund ist reinweiß (kein Grau, keine Textur)
- [ ] Nur eine Kernaussage pro Bild
- [ ] Handgezeichneter Stil erkennbar
- [ ] Deutsche Annotation vorhanden (falls vorgesehen)
- [ ] Weißraum macht mind. 40% des Bildes aus

## K.O.-Kriterien

- [ ] Sieht aus wie eine PowerPoint-Folie → ablehnen
- [ ] Zu viel Text im Bild → ablehnen
- [ ] LUMI fehlt oder ist kaum sichtbar → ablehnen
- [ ] Bunter Hintergrund → ablehnen
- [ ] Mehr als eine Hauptaussage → ablehnen
- [ ] Realistischer Illustrationsstil → ablehnen

## Bei Ablehnung

Prompt anpassen und neu generieren.
Maximal 3 Versuche pro Bild – dann Kompositionsmuster wechseln.
