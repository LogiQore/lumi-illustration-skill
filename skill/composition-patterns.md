# Kompositions-Muster

## Verfügbare Bildstrukturen

Wähle pro Bild EINE Struktur:

### 1. Aktion
LUMI führt etwas aus. Tippt, hält, trägt, schiebt.
*Einsatz:* Prozesse, Handlungen, Schritte

### 2. Vorher → Nachher
Links: Problem / Rechts: Lösung. LUMI steht in der Mitte oder zeigt.
*Einsatz:* Transformationen, Vergleiche

### 3. Metapher
LUMI interagiert mit einem überdimensionalen Objekt als Symbol.
*Einsatz:* abstrakte Konzepte greifbar machen

### 4. Zustandskarte
LUMI in verschiedenen Posen zeigt verschiedene Zustände.
*Einsatz:* Emotionen, Optionen, Varianten

### 5. Workflow
LUMI geht einen Pfad entlang mit Stationen.
*Einsatz:* Schritt-für-Schritt-Abläufe

### 6. Spotlight
LUMI beleuchtet oder zeigt auf einen Begriff oder ein Objekt.
*Einsatz:* Definitionen, Erklärungen

## Formatregeln

| Format | Verhältnis | Haupteinsatz |
|---|---|---|
| Instagram Post | 1:1 | Feed-Content |
| Instagram Story | 9:16 | Story/Reel-Intro |
| Pinterest Pin | 2:3 | Pinterest |
| YouTube Thumbnail | 16:9 | YouTube |

## Goldene Regel

Erfinde für jedes Bild eine NEUE physische Metapher.
Kopiere keine Muster aus Beispielbildern.
